Procedure to Create and View a Wedsite on the Production Sever

  1. Create a new file: gedit filename.html.
  2. Copy and paste HTML format into the new window that pop up.
  3. Open a new tab in the terminal and run the python:python -m SimpleHTTPServer.
  4. Go to the internet and type in 'localhost:8000/filename.html'into the url bar.
  5. Then start editing in the HTML format and refresh the page to see the change.
  6. For decoration open a new tab in terminal and type in gedit fliename.css.
  7. A new window will pop up and start editing with the CSS format.
  8. When done do git add, git commit -m, git push.
  9. Go back to terminal and type in: sftp jamie2215@cmis-hsteacher, enter and say yes to the question.
  10. Enter the password(github), type in cd public_html/ and put fliename.html/fliename.css. one at a time.
  11. To view the wedsite in the production server, go the url bar and type in: http://cmis-hsteacher/~(name)/(filename).
